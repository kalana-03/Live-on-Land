Content Page


Technical Description


1. HTML Structure


  • Navigation (<nav>): Responsive sidebar with SVG icons for toggling.
  • Intro Section (#intro): Background image, <h1>, paragraph, and navigation links.
  • Content Sections (.content): <h2> headings with structured <div class="cards"> for Causes, Impact, and Solutions.
  • Go Top Button (.top): Fixed-position <a href="#intro"Go top a> for easy navigation.
  • Footer (<footer>): Multiple <div class="footerContent"> blocks for links and info.

2. CSS Styling


  • Layout: content uses max-width: 1200px; with box-shadow.
  • Text: text-shadow on headings; font-size: 1.1rem; for readability.
  • Cards: display: flex; for alignment; hover effect lifts card.
  • Go Top Button: position: fixed; with a styled button.

Accessibility


  • Semantic HTML: <nav> <header> <footer> for screen reader support.
  • Focus Management: Hover and focus effects enhance interactivity.

Challenges and Lessons Learned


Challenges and Resolutions


-Technical Difficulties: Issues with coding, website responsiveness, and hosting were resolved through research, team collaboration, and seeking external resources.


-Team Coordination: Differing schedules and work styles were managed by assigning clear roles, conducting regular meetings, and utilizing project management tools.


-Content Accuracy: Ensuring credible and well-researched content was achieved by relying on reputable sources such as marine conservation organizations.


-Design and User Experience:Balancing aesthetics and functionality was addressed by following UI/UX principles and testing the website for accessibility.


-Time Management: Task division, scheduling, and deadline setting helped maintain project efficiency.


Lessons Learned


- Effective communication and teamwork are essential for success.


- Flexibility and adaptability help in overcoming unforeseen challenges.


- Research and credibility are crucial for informative content.


- User-centered design enhances engagement and accessibility.


- Proper planning and organization improve efficiency and productivity..
